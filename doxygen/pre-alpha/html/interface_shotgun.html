Return information about shotgun entities from the server.

Parameters:
entityTypeAn NSString specifying the type of entity to return.
filtersAn NSArray or NSDictionary corresponding to the valid filter values from the python API (or an NSString that is well formed JSON describing the same value).
fieldsAn NSArray of NSString that specifies what fields to return (or an NSString that is well formed JSON describing the same value).
orderAn NSArray of NSDictionary that specifies what fields to sort by (or an NSString that is well formed JSON describing the same value).
filterOperatorA string that controls whether to join filters as an 'and' (when filterOperator is "all") or as an 'or' (when filterOperator is anything else).
limitAn NSUInteger. Specifies the max number of entities to return.
pageAn NSUInteger. When specified will return page page of the results.
retiredOnlyA BOOL. Return retired entities if YES. Un-retired entities otherwise.
Returns:
A ShotgunRequest whose response is an NSArray of ShotgunEntity objects that match the filters.
